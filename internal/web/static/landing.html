How it works

A real engineer's loop — keep scrolling to sail it.

1 watch

Watches your board

The daemon polls Jira, GitHub Issues, or tasks you create locally — picks one ticket per cycle and starts the loop. No webhooks to host, no CI to wire up.

2 confirm

You confirm the repo

Before anything runs, goon asks which repository this ticket belongs to. Nothing executes in a checkout you didn't approve.

approval gate — your call
3 approve

You approve the plan

It triages the ticket into concrete steps and shows you the plan first. Edit steps, reject for a re-plan, or approve — one click from the dashboard or Telegram.

approval gate — your call
4 execute

It does the work

The agent codes inside your local checkout, runs your tests, and verifies the result — every shell command screened by a safety validator first.

5 ship

A pull request and a ping

goon pushes a branch, opens the PR on GitHub, GitLab, or Bitbucket, and notifies you on Telegram. Review the diff right in the dashboard before you merge.

Built to ship

Everything aboard a single binary.

No runtime, no containers, no external dependencies — drop one Go binary next to your repos and it gets to work.

Zero dependencies

Pure Go standard library. One binary, one workflow.json, and a folder of repos — that's the whole install.

Approval gates

Repo choice and plan approval pause the pipeline until you answer. Auto-approve flags exist when you're ready to trust it.

Any LLM provider

OpenAI, Anthropic, Gemini, or local Ollama — switch with one env var, or route each role (plan, code, review, chat) to a different model.

Live dashboard

Watch workflows advance stage by stage, answer gates, review diffs, edit pipelines, and track token usage — light and dark.

Telegram crew channel

Approve plans, ask questions, run commands, and get PR notifications from your phone — the whole loop works away from your desk.

Learns your project

Every run distills durable notes — repo maps, conventions, past decisions — so the next ticket starts smarter than the last.

Code in the browser

Pick a directory and run an agentic coding session right in the dashboard — read, edit, run commands, stream each step live. Claude Code in a tab.

Set sail in one line.

Install the binary, point it at your board and your LLM key, and the first ticket gets picked up on the next poll.

go install github.com/harisaginting/goon@latest
